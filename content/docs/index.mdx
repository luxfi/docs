# Welcome to Lux Network

Lux Network is a post-quantum secure, multi-consensus blockchain platform designed for the future of decentralized applications.

## Key Features

- **Post-Quantum Security**: Built with post-quantum cryptographic primitives
- **Multi-Consensus**: Support for multiple consensus mechanisms
- **High Performance**: Scalable architecture for enterprise applications
- **EVM Compatible**: Full Ethereum compatibility for easy migration

## Getting Started

```bash
# Install the Lux CLI
npm install -g @lux/cli

# Initialize a new project
lux init my-project

# Start development
cd my-project && lux dev
```
