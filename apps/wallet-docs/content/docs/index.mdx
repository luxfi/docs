Hierarchical deterministic wallet for Lux Network with support for all chain types and hardware wallets.

## Features

| Feature | Description |
|---------|-------------|
| HD Derivation | BIP-44 paths for P/X/C chains |
| Multi-Chain | UTXO (X), Platform (P), EVM (C) |
| Ledger | Hardware wallet signing |
| Multi-Sig | Threshold wallets |
