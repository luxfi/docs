Move assets between Lux chains, Ethereum, Bitcoin, and other chains using MPC threshold signatures.

## Flow

```
User ──► Bridge UI ──► Bridge API ──► MPC Nodes ──► Destination Chain
                                         │
                                    HSM Attestation
```

## Supported Chains

| Chain | Type | Assets |
|-------|------|--------|
| Lux C-Chain | EVM | LUX, WLUX, tokens |
| Zoo | L2 EVM | ZOO, tokens |
| Hanzo | L2 EVM | HANZO, tokens |
| Ethereum | EVM | ETH, ERC-20 |
| Bitcoin | UTXO | BTC |
