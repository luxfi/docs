## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| POST | `/v1/transfer` | Initiate transfer |
| GET | `/v1/transfer/:id` | Get transfer status |
| GET | `/v1/transfers` | List transfers |
| GET | `/v1/chains` | Supported chains |
