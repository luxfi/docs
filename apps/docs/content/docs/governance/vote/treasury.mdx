## Treasury Governance

The DAO treasury is managed through governance proposals. All fund movements require:

1. Formal proposal with detailed budget
2. Community discussion period (3+ days)
3. On-chain vote meeting quorum
4. Timelock delay before execution

## Multi-Sig

Emergency operations use a 4/7 multi-sig council with transparent on-chain execution.
