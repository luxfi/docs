The Lux CEX (`github.com/luxfi/cex`) is an institutional-grade Alternative Trading System (ATS) with a CLOB matching engine, multi-protocol gateway, and comprehensive compliance infrastructure.

## Regulatory Coverage

| Framework | Coverage |
|-----------|----------|
| **FINRA** | Rule 5310 (Best Execution), OATS reporting |
| **SEC** | ATS-N filing, CAT reporting |
| **MiFID II** | Best execution, transaction reporting |
| **Sanctions** | OFAC, UN, EU sanctions screening |
| **AML/KYC** | 30+ jurisdictions, PEP/EDD screening |

## Architecture

```
Clients → Gateway (REST/FIX/WS/ZAP) → Pre-Trade Compliance
              ↓                              ↓
         Matching Engine (CLOB)      Sanctions/PEP Check
              ↓                              ↓
         Settlement                  Post-Trade Surveillance
              ↓                              ↓
         Custody                     Regulatory Reporting
```

## Protocols

| Protocol | Port | Use Case |
