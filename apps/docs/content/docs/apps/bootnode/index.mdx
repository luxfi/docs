Lux Cloud provides unified APIs for interacting with blockchain networks. Deploy as lux.cloud, web3.hanzo.ai, or your own brand.

## APIs

| API | Description |
|-----|-------------|
| **RPC** | Multi-chain JSON-RPC proxy with load balancing |
| **WebSockets** | Real-time blockchain subscriptions |
| **Token API** | ERC-20/721/1155 balances and metadata |
| **NFT API** | Collections, ownership, media |
| **Webhooks** | Event-driven notifications |
| **Transfers** | Transaction history and analytics |

## White-Label

Lux Cloud supports white-label deployment for different brands. Set `NEXT_PUBLIC_BRAND` to customize.
