## Brands

| Brand | Domain | Config |
|-------|--------|--------|
| Lux Cloud | lux.cloud | Default |
| Hanzo Web3 | web3.hanzo.ai | `NEXT_PUBLIC_BRAND=hanzo` |
| Zoo Labs | web3.zoo.ngo | `NEXT_PUBLIC_BRAND=zoo` |
| Bootnode | bootno.de | `NEXT_PUBLIC_BRAND=bootnode` |

Brand is auto-detected from domain or set via environment variable.
