## Overview

Lux SDK is the **official Go SDK** for building and managing Lux networks — unified interface wrapping CLI, netrunner, and node APIs. Supports all 5 chain types.

## Quick reference

| Item | Value |
|------|-------|
| Module | `github.com/luxfi/sdk` |
| Go | 1.26.1 |

## Chain Support

| Chain | Type | Operations |
|-------|------|-----------|
| P-Chain | Platform | Staking, validators, L1 creation |
| X-Chain | Exchange | UTXO transfers, asset creation |
| C-Chain | Contract | EVM transactions, contract deployment |
| Q-Chain | Quantum | PQ-specific operations |
| M-Chain | MPC | Multi-party custody |

## Capabilities

- Network management (start, stop, health)
- L1/L2/L3 blockchain creation
- Staking and validator management
- Cross-chain transfers (Warp messaging)
- Wallet management
- Custom VM deployment
- WASM support

## Auto-Selection

SDK auto-selects the best available backend:
1. CLI (if installed)
2. Netrunner (if running)
3. Direct node APIs
4. Built-in fallback

## API Examples

### Node Client

```go
import "github.com/luxfi/sdk/client"

c := client.New("http://localhost:9630")

// Node info
nodeID, _ := c.Info().GetNodeID(ctx)
peers, _ := c.Info().Peers(ctx)
fmt.Printf("Node: %s, Peers: %d\n", nodeID, len(peers))

// P-Chain: Validators
validators, _ := c.Platform().GetCurrentValidators(ctx, nil)

// Health
health, _ := c.Health().Health(ctx)
```

### Wallet Operations

```go
import "github.com/luxfi/sdk/wallet"

w, _ := wallet.New(ctx, wallet.Config{
    URI:       "http://localhost:9630",
    NetworkID: 1, // mainnet
})

// Get balances
balance, _ := w.P().GetBalance(ctx)
fmt.Printf("P-Chain: %d nLUX\n", balance)

// Transfer on X-Chain
txID, _ := w.X().Transfer(ctx, wallet.TransferArgs{
    To:      "X-lux1...",
    Amount:  1_000_000_000, // 1 LUX
    AssetID: "LUX",
})
```

### Transaction Building

```go
import "github.com/luxfi/sdk/tx"

// Add validator (stake)
stakeTx, _ := tx.NewAddValidatorTx(tx.AddValidatorTxArgs{
    NodeID:      nodeID,
    Start:       time.Now().Add(1 * time.Minute),
    End:         time.Now().Add(365 * 24 * time.Hour),
    StakeAmount: 2000 * units.Lux, // 2000 LUX minimum
    RewardAddress: rewardAddr,
})
```

### C-Chain (EVM)

```go
import "github.com/luxfi/evm/ethclient"

ethClient, _ := ethclient.Dial("http://localhost:9630/ext/bc/C/rpc")
block, _ := ethClient.BlockNumber(ctx)
```
