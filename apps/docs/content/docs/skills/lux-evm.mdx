## Overview

Lux EVM is the **unified Go-based EVM execution engine** for both C-Chain and L1/L2 Subnet EVM chains. The former `coreth` codebase has been **merged into this repo** — it is not a separate dependency.

## Quick reference

| Item | Value |
|------|-------|
| Module | `github.com/luxfi/evm` |
| Version | v0.8.35+ |
| Go | 1.26.1 |
| C-Chain RPC | `http://<node>:9630/ext/bc/C/rpc` |
| Chain ID | 96369 (mainnet), 96370 (testnet) |
| Luxd compat | v1.12.0 through v1.13.3 (protocol versions 38-42) |

## Hard requirements

1. ALWAYS use `github.com/luxfi/evm` — NEVER `github.com/luxfi/coreth` (deprecated, merged here)
2. ALWAYS use `github.com/luxfi/geth` — NEVER `go-ethereum`

## Migration from coreth

```go
// OLD (deprecated):
import "github.com/luxfi/coreth/ethclient"

// NEW (correct):
import "github.com/luxfi/evm/ethclient"
```

All `coreth` imports become `evm` imports. The API is identical.

## Two Modes

### C-Chain Mode
- Atomic transactions for X-Chain/C-Chain cross-chain transfers
- Shared memory with other chains on the same node
- Integrated with P-Chain validator set

### Subnet EVM Mode
- Configurable fees, gas limits, and precompiles
- No atomic transactions (uses Warp messaging instead)
- Independent validator set per subnet

## One-file quickstart

### EVM RPC

```bash
# Get block number
curl -X POST http://localhost:9630/ext/bc/C/rpc \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"eth_blockNumber","params":[]}'

# Get balance
curl -X POST http://localhost:9630/ext/bc/C/rpc \
  -d '{"jsonrpc":"2.0","id":1,"method":"eth_getBalance","params":["0x...","latest"]}'

# Send transaction
curl -X POST http://localhost:9630/ext/bc/C/rpc \
  -d '{"jsonrpc":"2.0","id":1,"method":"eth_sendRawTransaction","params":["0x..."]}'
```

### Go Client

```go
import "github.com/luxfi/evm/ethclient"

client, _ := ethclient.Dial("http://localhost:9630/ext/bc/C/rpc")
blockNum, _ := client.BlockNumber(ctx)
balance, _ := client.BalanceAt(ctx, addr, nil)
```

### Custom Precompile

```go
import "github.com/luxfi/evm/precompile/contract"

var Module = &contract.Module{
    Address:   common.HexToAddress("0x0300000000000000000000000000000000000001"),
    Configure: configure,
}

func configure(cfg *contract.StatefulPrecompileConfig) {
    cfg.AddFunction("myFunction", myFunctionHandler)
}
```
