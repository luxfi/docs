## Overview

Lux Governance covers **on-chain voting, treasury management, and DAO infrastructure** for the Lux Network. The voting platform is published as `@pars/vote` (under Pars organization, not @luxfi directly).

## Components

| Component | Repo | Purpose |
|-----------|------|---------|
| Vote | `github.com/luxfi/vote` | Web voting platform (`@pars/vote`) |
| Treasury | `github.com/luxfi/treasury` | Treasury management |
| DAO | `github.com/luxfi/dao` | DAO infrastructure |

## Proposals

Governance changes are tracked via **LPs** (Lux Proposals) at `github.com/luxfi/lps`:
- **1xxx**: P-Chain proposals (validators, staking)
- **2xxx**: C-Chain proposals (EVM, precompiles)
- **3xxx**: X-Chain proposals (assets)
- **4xxx**: Network proposals (consensus, P2P)
