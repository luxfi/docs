## Overview

Ringtail is a pure Go implementation of the **Ringtail** post-quantum threshold signature scheme from [eprint.iacr.org/2024/1113](https://eprint.iacr.org/2024/1113). It provides practical 2-round lattice-based threshold signatures from standard LWE (Learning With Errors) assumptions, using the `luxfi/lattice` library for ring polynomial operations.

This is a critical building block for Lux's post-quantum consensus -- enabling t-of-n threshold signing without relying on classical assumptions (RSA, ECDSA) that are vulnerable to quantum computers.

## Hard requirements

1. ALWAYS use `github.com/luxfi/*` packages -- NEVER `go-ethereum` or `ava-labs`
2. NEVER use EWOQ keys -- generate fresh keys
3. Requires `github.com/luxfi/lattice/v7` for ring operations

## Scheme Parameters

From `sign/config.go`:

| Parameter | Value | Description |
|-----------|-------|-------------|
| LogN | 8 | Ring dimension N = 256 |
| Q | `0x1000000004A01` | 48-bit NTT-friendly prime |
| M | 8 | Public key rows |
| N | 7 | Secret key dimension |
| Kappa | 23 | Security parameter |
| KeySize | 32 (256 bits) | Key size |
| SigmaE | 6.108 | Error distribution width |
| SigmaStar | ~2^37.3 | Masking distribution width |

## Protocol Flow

```
1. GEN (Trusted Dealer)
   - Generate secret key shares via Shamir
   - Distribute seeds and MAC keys to parties

2. SIGN ROUND 1 (Each party)
   - Compute commitment D_i and MAC tags
   - Broadcast D_i to all parties

3. SIGN ROUND 2 (Each party)
   - Verify MACs from all parties
   - Compute partial signature z_i
   - Send z_i to combiner

4. FINALIZE (Combiner)
   - Aggregate partial signatures
   - Output final signature (z, Delta)

5. VERIFY (Anyone)
   - Verify signature against public key
```

## One-file quickstart

### Build

```bash
git clone https://github.com/luxfi/ringtail.git
cd ringtail
make build
# Output: bin/ringtail
```

### Run local simulation

```bash
# Local single-machine simulation (no networking)
./bin/ringtail l 1 3
# Args: l=local, 1=iterations, 3=parties
```

### Run distributed (P2P)

```bash
# Party 0 (trusted dealer + signer)
./bin/ringtail 0 1 3

# Party 1 (combiner + signer)
./bin/ringtail 1 1 3

# Party 2 (signer)
./bin/ringtail 2 1 3
```

### Run tests

```bash
make test
# Or with race detector:
make test-race
```

### Benchmarks

```bash
make bench
# CPU profiling:
make bench-cpu
# Memory profiling:
make bench-mem
```

## API Usage

```go
import (
    "github.com/luxfi/ringtail/sign"
    "github.com/luxfi/ringtail/primitives"
    "github.com/luxfi/lattice/v7/ring"
    "github.com/luxfi/lattice/v7/utils/sampling"
)

// Setup ring
r, _ := ring.NewRing(1<<sign.LogN, []uint64{sign.Q})
prng, _ := sampling.NewKeyedPRNG(randomKey)
uniformSampler := ring.NewUniformSampler(prng, r)

// Create party
party := sign.NewParty(partyID, r, r_xi, r_nu, uniformSampler)

// Trusted dealer: generate key shares
A, skShares, seeds, MACKeys, b := sign.Gen(r, r_xi, uniformSampler, dealerKey, lagrangeCoeffs)

// Sign Round 1
D, MACs := party.SignRound1(A, sessionID, prfKey, T)

// Sign Round 2 (after receiving all D and MACs)
valid, DSum, hash := party.SignRound2Preprocess(A, b, D, MACs, sessionID, T)
z := party.SignRound2(A, b, DSum, sessionID, message, T, prfKey, hash)

// Finalize (combiner collects all z)
_, sig, Delta := party.SignFinalize(zMap, A, b)

// Verify
verified := sign.Verify(r, r_xi, r_nu, sig, A, message, b, party.C, Delta)
```

## Consensus Integration

Ringtail provides the post-quantum threshold signature primitive used in Lux's Quasar consensus for the second-round lattice-based PQ certificate. The `luxfi/consensus` module depends on this for quantum-resistant finality.

## Troubleshooting

| Issue | Cause | Solution |
|-------|-------|----------|
| MAC verification failed | Network latency or corruption | Check TCP connectivity between parties |
| NTT panic | Ring dimension mismatch | Ensure LogN matches across all parties |
| Slow benchmarks | Large party count | Start with 3 parties, scale up |
| Connection refused | Party not started | Start all parties before timeout |
