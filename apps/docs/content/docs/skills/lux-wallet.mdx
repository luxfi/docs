## Overview

Lux Wallet is a **multi-platform hardware wallet application** — `@luxwallet/monorepo` supporting Desktop, Web, Mobile (iOS/Android), and Browser Extension from a single codebase. Fork of OneKey, extensively customized for Lux chain support.

## Quick reference

| Item | Value |
|------|-------|
| Repo | `github.com/luxfi/wallet` |
| Package | `@luxwallet/monorepo` v1.0.0 |
| Package manager | Yarn 4.1.0 (Berry) — NOT pnpm |
| Workspace packages | `@onekeyfe/*`, `@luxwallet/*` |

## Chain Support

Lux native chains plus multi-chain:
- **Lux**: P-Chain, X-Chain, C-Chain, Q-Chain, M-Chain
- **EVM**: Ethereum and all EVM-compatible chains
- **Polkadot**: Substrate-based chains
- **Sui/Aptos**: Move-based chains
- **Cardano**: Via `@emurgo` libraries
- **Bitcoin**: Via `btcsuite`

## Hardware Wallet Support

Hardware wallet integration via `@onekeyfe/hd-*` SDK packages:
- OneKey Classic/Touch/Mini/Pro
- Ledger (via `github.com/luxfi/ledger-go`)

## Go SDK Wallet

```go
import (
    "github.com/luxfi/go-bip39"
    "github.com/luxfi/wallet"
)

// Generate mnemonic
mnemonic, _ := bip39.NewMnemonic(256) // 24 words

// Create wallet
w, _ := wallet.FromMnemonic(mnemonic)

// Get addresses for each chain
xAddr := w.X().Address()  // X-lux1...
pAddr := w.P().Address()  // P-lux1...
cAddr := w.C().Address()  // 0x...
```
