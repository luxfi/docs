## Overview

Lux UI provides **React components, chart visualizations, status dashboards, and terminal tools** for the Lux ecosystem.

## Components

| Component | Path | Purpose |
|-----------|------|---------|
| ui/ | ``github.com/luxfi/ui`` | @lux/ui React components |
| charts/ | ``github.com/luxfi/charts`` | Chart visualizations |
| status/ | ``github.com/luxfi/status`` | Status monitoring dashboard |
| tui/ | ``github.com/luxfi/tui`` | Terminal UI tools (Go) |
