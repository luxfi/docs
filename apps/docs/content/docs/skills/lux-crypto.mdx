## Overview

Lux Crypto provides the **cryptographic primitives** for the Lux Network — BLS threshold signatures, SLIP-10 HD wallets, secp256k1, KZG commitments, and post-quantum via gnark-crypto.

## API

### BLS Signatures

```go
import "github.com/luxfi/crypto/bls"

// Generate key pair
sk, _ := bls.NewSecretKey()
pk := sk.PublicKey()

// Sign and verify
msg := []byte("transaction data")
sig := bls.Sign(sk, msg)
valid := bls.Verify(pk, msg, sig)

// Threshold signatures (t-of-n)
shares := bls.Split(sk, threshold, total)
partialSigs := make([]bls.Signature, threshold)
for i := 0; i < threshold; i++ {
    partialSigs[i] = bls.Sign(shares[i], msg)
}
combinedSig := bls.Combine(partialSigs)
```

### HD Key Derivation (SLIP-10)

```go
import "github.com/luxfi/crypto/keychain"

kc, _ := keychain.NewFromSeed(seed)
child, _ := kc.Derive(keychain.DerivationPath{
    44,   // purpose
    9005, // Lux coin type
    0,    // account
    0,    // change
    0,    // index
})
```
