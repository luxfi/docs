## Overview

Lux Bitcoin is a **static marketing site** for Lux Network's Bitcoin integration. Built with Next.js 15, Tailwind CSS, and Radix UI primitives, it uses MDX for content and pulls from the `@luxfi/ui` and `@hanzo/ui` component libraries. Content covers Lux's energy-efficient approach to Bitcoin, ESG commitments, and global expansion.

## When to use

- Modifying the Lux Bitcoin landing page content or design
- Updating marketing copy about Lux Network's Bitcoin integration
- Working with the `@luxfi/ui` or `@luxfi/data` component libraries

## Project structure

```

  public/              # Static assets
  src/
    app/               # Next.js App Router pages
    content/           # MDX and TSX content blocks
      about-lux-energy.mdx     # Energy efficiency content
      commitment-to-esg.mdx    # ESG commitment content
      global-expansion.mdx     # Global expansion content
      powering.tsx             # Powering section
      index.ts                 # Content barrel export

    metadata.ts        # Site metadata and SEO
    site-def.ts        # Site definition constants
  next.config.js       # Next.js configuration
  postcss.config.js    # PostCSS with Tailwind
  tailwind.config.ts   # Tailwind CSS configuration
  tsconfig.json        # TypeScript configuration
  svgr.next.config.js  # SVG-as-React-component config
```

## One-file quickstart

```bash
git clone https://github.com/luxfi/bitcoin.git
cd bitcoin
pnpm install
pnpm dev
```

## Development

```bash
# Dev server
pnpm dev

# Production build
pnpm build

# Start production server
pnpm prod

# Type check
pnpm tc

# Clean build artifacts
pnpm cn

# Full clean (remove node_modules)
pnpm clean
```

## Content architecture

The site uses MDX files in `src/content/` for marketing copy, paired with TSX files that define layout and component structure. Content is imported via a barrel `index.ts` and rendered through custom `mdx-components.tsx` overrides.

Content sections:
- **About Lux Energy** -- Energy efficiency of Lux Bitcoin
- **Commitment to ESG** -- Environmental, social, governance positioning
- **Global Expansion** -- Network growth and partnerships
- **Powering** -- What Lux Bitcoin powers
