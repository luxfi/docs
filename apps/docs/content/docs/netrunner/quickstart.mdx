## Start the Server

Netrunner runs as a gRPC server that manages node processes:

```bash
netrunner server --port=":8080" --grpc-gateway-port=":8081"
```

## Start a Network

In a separate terminal, start a 5-node network:

```bash
netrunner control start \
  --number-of-nodes=5 \
  --node-path=/path/to/luxd
```

## Verify Health

```bash
netrunner control health
```

Expected output shows all nodes as healthy with bootstrapped chains.

## Check Status

```bash
netrunner control status
```

Lists all running nodes with their endpoints, NodeIDs, and chain status.

## Run a Quick Benchmark

```bash
netrunner bench --tps --tx-count 1000
```

## Save a Snapshot

Save the current state for later restoration:

```bash
netrunner control save-snapshot my-first-snapshot
```

## Stop the Network

```bash
netrunner control stop
```

## Restore from Snapshot

```bash
netrunner control load-snapshot my-first-snapshot
```
