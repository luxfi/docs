Lux Proposals (LPs) define the standards, protocols, and governance processes for the Lux Network. All proposals are tracked at [lps.lux.network](https://lps.lux.network).

## Categories

| Range | Category | Description |
|-------|----------|-------------|
| 0-99 | **Network** | Architecture, governance, framework |
| 100-299 | **Tokens** | Token standards (LRC-20, LRC-721, LRC-1155) |
| 300-599 | **Consensus** | Quasar, Photon, finality protocols |
| 600-2999 | **Chains** | Specialized chain specifications |
| 3000-3999 | **Privacy** | Z-Chain, FHE, ZK proofs |
| 4000-4999 | **DeFi** | DEX, AMM, derivatives |
| 5000-5999 | **AI/Attestation** | A-Chain, TEE, LLM integration |
| 6000-6999 | **Infrastructure** | Warp, DA, indexing |
| 9000+ | **Research** | Experimental proposals |
