Lux Network uses multiple layers of cryptographic infrastructure for key management, threshold signing, and encrypted computation.

| Module | Description |
|--------|-------------|
| [HSM](/docs/crypto/hsm) | Hardware security module integration (AWS, GCP, Azure, Zymbit, ML-DSA) |
| [KMS](/docs/crypto/kms) | Enterprise key management, secrets, PKI, K8s operator |
| [MPC](/docs/crypto/mpc) | Threshold signing (CGGMP21, FROST, BLS, Ringtail) |
| [FHE](/docs/crypto/fhe) | Fully homomorphic encryption (TFHE, CKKS, coprocessor) |
| [Lattice](/docs/crypto/lattice) | Lattice-based post-quantum cryptography |
