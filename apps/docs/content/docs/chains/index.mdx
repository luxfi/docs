Lux Network supports multiple chain types through its virtual machine architecture. Each VM is purpose-built for specific workloads.

## Primary Chains

| Chain | VM | Chain ID | Purpose |
|-------|-----|----------|---------|
| **C-Chain** | EVM | 96369 | Smart contracts, DeFi |
| **P-Chain** | PlatformVM | — | Validators, staking, subnets |
| **X-Chain** | ExchangeVM | — | UTXO asset transfers |
