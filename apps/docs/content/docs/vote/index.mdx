The Lux voting app provides a streamlined interface for DAO governance participation.

## Links

- [lux.vote](https://lux.vote) — Main voting interface
- [DAO documentation](/dao) — Full DAO governance docs
- [Lux Proposals](https://lps.lux.network) — Standards and proposals

## Participation

1. Connect wallet with vLUX (voting-escrowed LUX)
2. Browse active proposals
3. Vote (For / Against / Abstain)
4. Delegate voting power (optional)
