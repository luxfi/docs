---
title: Deploy an L2
description: Create and deploy an L2 EVM chain
icon: Layers
---

## Create

```bash
lux chain create mychain --evm
```

Follow the prompts to configure gas, fees, and genesis allocations.

## Deploy Locally

```bash
lux chain deploy mychain --local
```

## Deploy to Testnet

```bash
lux chain deploy mychain --testnet
```

Requires funded keys on the P-Chain.
