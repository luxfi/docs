## Create a Local Network

```bash
lux network start
```

## Create a Chain

```bash
lux chain create mychain --evm
```

## Deploy the Chain

```bash
lux chain deploy mychain --local
```

## Send a Transaction

```bash
lux wallet transfer --to 0x... --amount 1 --chain mychain
```
