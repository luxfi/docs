## Script Install

```bash
curl -sSL https://cli.lux.network/install | sh
```

## Go Install

```bash
go install github.com/luxfi/cli@latest
```

## Homebrew

```bash
brew install luxfi/tap/lux
```

## Verify

```bash
lux --version
```
