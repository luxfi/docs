# dApp Explorer Proposals

The dApp Explorer lets you create governance proposals by interacting with any external dApp. Instead of manually encoding contract calls, you browse and interact with dApps directly, and the resulting transactions become proposal actions.

## How It Works

1. Open the **dApp Explorer** from the proposal creation screen
2. Navigate to any supported dApp (DEX, lending protocol, NFT marketplace)
3. Configure the transaction as you normally would
4. Instead of executing immediately, the transaction is captured as a proposal action
5. Submit the proposal for governance voting

## Supported Actions

Any on-chain transaction can be proposed through the dApp Explorer:

- **DEX Swaps** - Swap treasury assets on decentralized exchanges
- **Lending** - Deposit or withdraw from lending protocols
- **NFT Operations** - Purchase, sell, or manage NFTs
- **Staking** - Stake tokens in external protocols
- **Contract Interactions** - Call any smart contract function

## Creating a dApp Explorer Proposal

1. Go to **Proposals** and click **New Proposal**
2. Select **dApp Explorer** as the proposal type
3. Enter the dApp URL or select from the directory
4. Interact with the dApp to build your transaction
5. Review the captured action details
6. Add a title, description, and rationale
7. Submit the proposal

## Security Considerations

- All captured transactions are displayed in human-readable format before submission
- Proposal reviewers can verify the exact contract calls
- Timelocked execution ensures community review
