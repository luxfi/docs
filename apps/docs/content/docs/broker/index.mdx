The Lux Broker (`github.com/luxfi/broker`) provides institutional-grade smart order routing across 16+ trading venues spanning crypto, equities, FX, and custody.

## Supported Venues

| Category | Venues |
|----------|--------|
| **Crypto** | Coinbase, Binance, Kraken, Gemini, SFOX, FalconX |
| **Equities** | Alpaca, Interactive Brokers, Tradier |
| **Custody** | Fireblocks, BitGo |
| **Stablecoins** | Circle (USDC) |
| **FX** | CurrencyCloud (35 pairs), LMAX |
| **Market Data** | Polygon.io |
| **Payments** | Finix |

## Features

| Feature | Description |
|---------|-------------|
| **Smart Order Routing** | Fee-aware net-price optimization |
| **Order Splitting** | VWAP execution across venues |
| **Pre-Trade Risk** | Position limits, rate limits, concentration |
| **Best Execution** | FINRA Rule 5310 / MiFID II compliant |
| **Multi-Asset** | Crypto, equities, FX, options, futures, metals |
| **TWAP** | Time-weighted scheduling for large orders |
