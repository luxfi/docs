## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| POST | `/v1/orders` | Submit order with SOR |
| GET | `/v1/quotes` | Get multi-venue quotes |
| GET | `/v1/orders/:id` | Order status |
| GET | `/v1/executions` | Execution history |
| GET | `/v1/venues` | Connected venues |
| GET | `/v1/balances` | Cross-venue balances |
