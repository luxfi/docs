The Lux CLI provides tools for network management, subnet deployment, wallet operations, and validator control.

## Quick Install

```bash
curl -sSL https://cli.lux.network/install | sh
```

## Commands

| Command | Description |
|---------|-------------|
| `lux network` | Start, stop, and manage local networks |
| `lux subnet` | Create and deploy subnets |
| `lux key` | Manage keys and addresses |
| `lux wallet` | Transfer assets and check balances |
| `lux node` | Validator operations and health |
