## Certificate Lifecycle

1. **Create CA** — Root or intermediate certificate authority
2. **Issue Certificates** — Sign CSRs with configurable profiles
3. **Revoke** — CRL and OCSP support
4. **Renew** — Automatic renewal before expiry

## EST Protocol

Full EST (RFC 7030) support for automated certificate enrollment.
