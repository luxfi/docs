## Operator

The Lux Operator (`ghcr.io/luxfi/operator:v0.3.9`) manages indexer deployments via the `LuxIndexer` CRD.

## Namespace

All indexer instances run in `lux-explorer` namespace on the `lux-k8s` cluster.
