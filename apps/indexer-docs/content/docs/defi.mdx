The indexer has native support for indexing DeFi protocol events, providing structured data beyond raw transaction logs.

## Supported Protocols

### AMMs / DEXs

| Protocol | Versions | Features |
|----------|----------|----------|
| Uniswap | V2, V3 | Swaps, liquidity, positions |
| SushiSwap | V1, V2 | Swaps, farms |
| Curve | All | Stable swaps, gauges |
| Balancer | V2 | Weighted pools |
| PancakeSwap | V2, V3 | BSC + multi-chain |

### Lending

| Protocol | Versions | Features |
|----------|----------|----------|
| Aave | V2, V3 | Supply, borrow, liquidations |
| Compound | V2, V3 | cTokens, governance |
| Morpho | V1 | Peer-to-peer matching |
| Benqi | V1 | Lux-native lending |

### Perpetuals

| Protocol | Features |
|----------|----------|
| GMX | V1, V2 positions |
| Synthetix | Synth tracking |
| dYdX | Order book |

### NFT Marketplaces

## OHLCV Market Data

The indexer generates OHLCV (Open-High-Low-Close-Volume) candles for all indexed trading pairs, suitable for charting.
