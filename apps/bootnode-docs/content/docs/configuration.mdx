## Environment Variables

| Variable | Description |
|----------|-------------|
| `API_KEY_SALT` | Salt for API key hashing |
| `DATABASE_URL` | PostgreSQL connection |
| `REDIS_URL` | Redis for WebSocket state |
| `RPC_ENDPOINTS` | Upstream node endpoints |
| `NEXT_PUBLIC_BRAND` | Brand (lux/hanzo/zoo/bootnode) |
