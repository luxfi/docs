## Get Balances

```bash
curl https://api.lux.cloud/v1/tokens/balances?address=0x...
```

## Supported Standards

| Standard | Features |
|----------|----------|
| ERC-20 | Balance, transfers, metadata |
| ERC-721 | Ownership, metadata, media |
| ERC-1155 | Multi-token balances |
