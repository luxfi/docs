## Language Support

| Language | Package | Chains |
|----------|---------|--------|
| Go | `github.com/luxfi/sdk` | P, X, C, Subnets |
| TypeScript | `@luxfi/sdk` | P, X, C, Subnets |
| Python | `luxfi-sdk` | C-Chain, Subnets |
