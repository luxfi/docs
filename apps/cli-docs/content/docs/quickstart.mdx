## Create a Local Network

```bash
lux network start
```

## Create a Subnet

```bash
lux subnet create mysubnet --evm
```

## Deploy the Subnet

```bash
lux subnet deploy mysubnet --local
```

## Send a Transaction

```bash
lux wallet transfer --to 0x... --amount 1 --chain mysubnet
```
